# SCuM Programmer

Load this firmware onto an nRF52840-DK to turn it into a programmer for SCuM!

## SCuM?!?

The Single Chip micro-Mote, the 2x3mm2 single-chip standard-compatible Smart Dust chip, see https://www.crystalfree.org/
